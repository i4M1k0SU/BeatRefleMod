# BeatRefleMod

beat refleのMod

## 機能
- 判定の表示
- 総ノーツ数の表示
- FPSの表示

![screenshot](images/screenshot.jpg)

## 使い方

mono対応のDLL InjectorでInjectする

- Class name: `Loader`
- Method name: `Main` or `Init` or `Load`

### 例
- GH Mono Injector
- MInjector
- SharpMonoInjector
